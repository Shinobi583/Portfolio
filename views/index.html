The Ripple Effect

(GitHub: Blog Code) (Site: The Ripple Effect)

Built with no CMS or Front-End Framework, The Ripple Effect is my personal blog site. It is a Full-Stack application using Nodejs that connects to a database(MongoDB), utilizes Authentication, and follows the MVC architecture.
Currently, I'm using a free tier of Heroku to host. This means that if there is no activity on the website for 30 minutes, it will take a few extra seconds to load the website for the first time. After that, it will run like normal.

FancyTab

(GitHub: FancyTab) (Chrome Store: FancyTab)

A chrome extension that allows for a fancier version of a new tab! Change the tab's background, your name, view your city's weather, view your recent pages, and search using DuckDuckGo!
The Chrome Store version does not use the weather API due to the API key being registered under a free license for demonstration purposes.
If you would still like to try it out with the weather, follow the instructions in the Readme here.
